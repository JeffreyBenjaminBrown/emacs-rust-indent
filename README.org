* To hot-reload the code
Adjust the path here to your liking.

#+begin_src elisp
(progn
  (when (fboundp 'jbb-rust-indent-disable)
    (jbb-rust-indent-disable))
  (load-file "~/hodal/skg/rust-indent/jbb-rust-indent.el")
  (jbb-rust-indent-enable))
#+end_src
* How to read and update the tests
  tests-distilled.rs is the source of truth for tests.

  tests.el, which is less readable, is auto-generated from it.
